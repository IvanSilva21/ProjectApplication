# ProjectApplication

API REST para gerenciamento de produtos.

## Tecnologias
- Java 17
- Spring Boot
- Spring Data JPA
- H2 Database
- Swagger / OpenAPI

## Funcionalidades
- Cadastro de produtos
- Listagem
- Busca por ID
- Atualização
- Remoção

## Documentação
Swagger disponível em:
http://localhost:8080/swagger-ui.html
